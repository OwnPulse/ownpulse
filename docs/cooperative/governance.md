# Cooperative Governance Model

OwnPulse operates as a data cooperative under a nonprofit structure. Users collectively own and govern the platform.

See [ADR-0010](../decisions/0010-data-cooperative.md) for the full rationale.

## Structure

OwnPulse is a nonprofit cooperative (legal structure to be finalized with counsel). The cooperative owns the hosted infrastructure and the OwnPulse trademark. The codebase is AGPL-3.0 licensed and belongs to the community.

## Principles

1. **One member, one vote.** Every user with an active account is a member. Membership is not tiered. Financial contribution does not buy additional votes.

2. **User-elected board.** Board seats are elected by the membership. The founding team holds initial board seats with a transition plan to fully elected governance.

3. **Transparency.** Board meetings, financial statements, and major decisions are published to members. Compensation of any paid roles is disclosed.

4. **Open development.** All code is open source (AGPL-3.0). All architecture decisions are documented as ADRs. All roadmap decisions are discussed publicly.

## Revenue Model

- **Self-hosting:** Free. The code is AGPL-3.0 and runs on any Linux VPS.
- **Hosted cooperative instance:** Subscription covers infrastructure costs.
- **Research data marketplace (Phase 3):** Anonymized aggregate data is sold to researchers with user consent. Revenue split: 80% to consenting users (as hosting credits or charitable donations), 20% to cooperative operating costs.

## Decision Making

- Day-to-day technical decisions: made by contributors via PR review.
- Architecture decisions: documented as ADRs, reviewed by maintainers.
- Policy decisions (pricing, data sharing terms, partnerships): board vote, published to members.
- Constitutional changes (governance structure, membership terms): member vote with supermajority.

## Membership

- Any user with an active OwnPulse account (self-hosted or hosted) is a member.
- Members can vote in board elections and policy referenda.
- Members can propose agenda items for board meetings.
- Membership is free. There is no pay-to-play governance.
