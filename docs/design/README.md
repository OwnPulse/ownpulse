# Design System

This directory contains brand guidelines and design assets for OwnPulse.

## Contents

- [brand.md](brand.md) -- Brand guidelines: colors, typography, logo usage, and tone.

## Principles

The design system prioritizes clarity and data density over decoration. Health data dashboards need to communicate a lot of information without clutter. Interactions should be fast and predictable.

Additional design assets (wireframes, component specs) will be added as the UI matures.
